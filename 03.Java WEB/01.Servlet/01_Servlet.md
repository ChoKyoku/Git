## 3.Cookie 和 Session

cookie 和 session 都是用于在web中跟踪用户状态的技术。

### 3.1 Cookie

Cookie是存放在客户端浏览器上的小文本文件，由服务器发往浏览器，当用户再一次访问网站的时候，浏览器会自动的把cookie的内容发往服务器。比如cookie中可记住用户的登录状态或语言偏好等。Cookie是有一定的有效期限的，可以设置过期时间。

Cookie是按照键值对的方式存储一些字符串
